# Clone-App — RioCard Mais+

Projeto desenvolvido em **React Native** com **Expo** e **TypeScript**, criado como exercício de clonagem de interface e funcionalidades do aplicativo **RioCard Mais+**.

## 📱 Sobre o projeto

Este repositório reproduz a estrutura e o design do app RioCard Mais+, com o objetivo de praticar conceitos de desenvolvimento mobile, navegação entre telas e organização de componentes em React Native.

## 🚀 Tecnologias utilizadas

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [React Navigation](https://reactnavigation.org/) (native + native-stack)
- [Expo Vector Icons](https://icons.expo.fyi/)
- [React Native Safe Area Context](https://github.com/th3rdwave/react-native-safe-area-context)
- [React Native Screens](https://github.com/software-mansion/react-native-screens)

## 📋 Pré-requisitos

Antes de começar, você precisa ter instalado em sua máquina:

- [Node.js](https://nodejs.org/)
- [Git](https://git-scm.com/)
- [Expo Go](https://expo.dev/go) (aplicativo no celular) ou um emulador Android/iOS configurado

## 🔧 Instalação e execução

Clone o repositório:

```bash
git clone https://github.com/yasmimverissimo/Clone-App.git
```

Acesse a pasta do projeto:

```bash
cd Clone-App
```

Instale as dependências:

```bash
npm install
```

Inicie o projeto:

```bash
npm start
```

Após iniciar, um QR Code será exibido no terminal. Escaneie-o com o app **Expo Go** no seu celular ou use um dos comandos abaixo para abrir em um emulador específico:

```bash
npm run android   # Abre no emulador/dispositivo Android
npm run ios       # Abre no simulador iOS (necessário macOS)
npm run web       # Abre no navegador
```
